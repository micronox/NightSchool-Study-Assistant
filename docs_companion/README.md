# Docusaurus Companion

This folder is the presentation-layer companion for the NightSchool Study Assistant packet.

## Goal

Turn the packet into a guided, website-like reading experience:

- fast top-level orientation
- clear distinction between verified outcomes and planned work
- navigable links into architecture and evidence
- easy handoff to non-technical readers

## Suggested next step

After the GitHub repo exists, replace the placeholder repo URL in the docs with the final location:

`https://github.com/<owner>/NightSchool-Study-Assistant`
