### Gateway launcher wrappers are real and nearly identical

Commander wrapper:

- changes directory to `C:\Users\larry\.hermes\hermes-agent`
- sets `HERMES_HOME` to the Commander profile
- sets `HERMES_GATEWAY_DETACHED=1`

  - `pythonw.exe -m hermes_cli.main --profile commander gateway run`

Apollo wrapper:

- same structure
- `HERMES_HOME` points to Apollo profile
- launches:
  - `pythonw.exe -m hermes_cli.main --profile apollo gateway run`

Operational implication:

- both profiles depend on the same shared runtime install:
  `C:\Users\larry\.hermes\hermes-agent`
- if that shared runtime, its environment, or its launch assumptions changed,
  both profile gateways could fail together
