# Architecture Defense and Preproduction

This folder holds the two-layer packet for the NightSchool Study Assistant:

- `master_packet/` preserves the complete verified packet and provenance.
- `external_share_candidate/` is the cleaner external-facing handoff.

## Usage

- Use `master_packet/` as the internal source of truth.
- Use `external_share_candidate/` for first-pass review by external readers.
- Pull the executive narrative from the same documents when updating the docs site or PDF layer.
